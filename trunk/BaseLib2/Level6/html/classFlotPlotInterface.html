Detailed Description

Provides web plotting with zoom to BaseLib2 classes.

This class is capable of plotting data using using flot: http://code.google.com/p/flot/ It has to be extended by a signal provider implementing the GetSignalData method

The url arguments are:
SignalList=mandatory comma separated signal names
xxSignal=optional signal name for the xx axis
xxMultiplier=a multiplier for the xx values
yyMultiplier=a multiplier for the yy values
drawLines=true to draw the lines between the points (fast)
drawPoints=true to draw a circle around each point (slow)
Width=optional width of the chart (default 600)
Height=optional height of the chart (default 300)
example:http://localhost:8084/BROWSE/FlotPlot/?SignalList=CYCLETIME,TIMERRELATIVEUSECTIME&xxSignal=TIME&xxMultiplier=1e-6&yyMultiplier=1e6

It also expects to find the flot scripts in the URL http://.../FLOT_DIR/
